CTZN

My capstone project, CTZN is a web application designed to help citizens follow the progress of local, state, and federal legislation, and to help the user take actions related to it. Users are able to search for bills, add them to their queue, and set “action” reminders which prompt them to contact their representatives through various means of communication or attend an event or meeting. CTZN makes use of Ruby on Rails, AngularJS, and various public APIs, such as ProPublica’s Congress API and OpenStates API. I chose to build this app to streamline the process of following legislation and taking actions to support or oppose it.
